# Health Crystal Ball – AI Health Risk Predictor

## Project Description
The Health Crystal Ball is a machine learning-based AI application that predicts health risk based on lab values and demographic information. It helps users understand potential liver-related health risks by analyzing inputs such as bilirubin levels, liver enzymes, and protein values.

---
## Features
- Predicts low, medium, or high health risk using AI models.
- Uses Random Forest and XGBoost for prediction.
- Interactive Streamlit app for easy input and results.
- Includes trained model (best_model.pkl) and requirements.txt for running locally.
- PDF presentation (Health_Crystal_Ball.pdf) included to explain the project.

---

## Input Parameters

| Parameter                  | Description                                 |
|-----------------------------|---------------------------------------------|
| Age                         | User’s age in years                         |
| Gender                      | Male / Female                               |
| Total Bilirubin             | Liver function parameter                     |
| Direct Bilirubin            | Liver function parameter                     |
| Alkaline Phosphatase (ALP) | Liver enzyme level                           |
| ALT (SGPT)                  | Liver enzyme level                           |
| AST (SGOT)                  | Liver enzyme level                           |
| Total Proteins              | Blood protein level                          |
| Albumin                     | Blood protein level                          |
| Albumin/Globulin Ratio      | Albumin divided by Globulin                  |
